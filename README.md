# Online Sales Analysis

## Descriere
Un proiect Python pentru managingul de produse si cosul de cumparaturi al utilizatorului intr-un magazin online.

## Clase

### Product
Reprezinta un produs cu:
- nume
- pret
- cantitate

### ProductManager
Se ocupa de:
- adaugarea produselor
- aratarea produselor
- calcularea valorii inventarului
- eliminarea produselor

### Cart
Se ocupa de:
- adaugarea produselor in cos
- aratarea cosului
- calcularea pretului total al cosului
